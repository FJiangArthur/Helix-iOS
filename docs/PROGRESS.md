- Multi-backend transcription (Apple On-Device / Apple Cloud / OpenAI
  Transcription / OpenAI Realtime)

- 5 LLM providers (OpenAI, Anthropic, DeepSeek, Qwen, Zhipu)

- 4-tab UI (Home / Glasses / History / Settings), onboarding flow
