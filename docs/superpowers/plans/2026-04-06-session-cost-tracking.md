# Cumulative Session Cost Tracking Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add live cumulative session cost display, per-session DB persistence, history list display, and per-model breakdown (smart / light / transcription) so the user can see where money goes.

**Architecture:** Extend `ConversationCostTracker` with a `Stream<SessionCostSnapshot>` for live updates and add `modelRole` tagging to per-call recording. Add nullable `cost*UsdMicros` columns to the `Conversations` Drift table via a non-destructive migration (schema v3 -> v4). Surface live cost in the home screen recording bar pill and per-session cost in the history list cell with a tap-through breakdown sheet. All costs stored as integer micro-USD (`1 USD = 1_000_000`) to avoid float drift.

**Tech Stack:** Flutter 3.35+, Dart, Drift (SQLite), GetX, iOS 26 deployment target.

**Depends on:** Spec A for `modelRole` tagging on LLM calls (smart / light / transcription). This plan lands the plumbing; Spec A is responsible for actually passing `ModelRole.smart` vs `ModelRole.light` at the call sites in `conversation_engine.dart`. This plan defaults `null` -> treated as `smart` in aggregation.

## Phase 4 — `PricingRegistry` with light-model + Realtime-per-minute entries

Goal: central, typed price lookup. No UI coupling.

- [ ] **4.1 Create registry**
  - File: `lib/services/cost/pricing_registry.dart`
  - Contents (skeleton):
    ```dart
    class ModelPricing {
      const ModelPricing({
        this.inputPerMillionUsd,
        this.cachedInputPerMillionUsd,
        this.outputPerMillionUsd,
        this.audioInputPerMillionUsd,
        this.audioInputPerMinuteUsd,
      });
      final double? inputPerMillionUsd;
      final double? cachedInputPerMillionUsd;
      final double? outputPerMillionUsd;
      final double? audioInputPerMillionUsd;
      final double? audioInputPerMinuteUsd;
    }

    class PricingRegistry {
      PricingRegistry._();
      static final PricingRegistry instance = PricingRegistry._();

      // Source: https://openai.com/api/pricing/ (captured 2026-04-06)
      // NOTE: any rate change is a one-line edit here. Do not read from SharedPreferences.
      static const Map<String, Map<String, ModelPricing>> _table = {
        'openai': {
          // Smart models (Spec A default smart)
          'gpt-5.4': ModelPricing(inputPerMillionUsd: 1.25, cachedInputPerMillionUsd: 0.125, outputPerMillionUsd: 10.0),
          'gpt-5.4-mini': ModelPricing(inputPerMillionUsd: 0.25, cachedInputPerMillionUsd: 0.025, outputPerMillionUsd: 2.0),
          // Light models (Spec A default light)
          'gpt-5.4-nano': ModelPricing(inputPerMillionUsd: 0.05, cachedInputPerMillionUsd: 0.005, outputPerMillionUsd: 0.4),
          // Transcription
          'gpt-4o-mini-transcribe': ModelPricing(audioInputPerMillionUsd: 3.0),
          'gpt-4o-transcribe': ModelPricing(audioInputPerMillionUsd: 6.0),
          'whisper-1': ModelPricing(audioInputPerMinuteUsd: 0.006),
          // OpenAI Realtime — billed per audio minute
          'gpt-4o-mini-realtime': ModelPricing(audioInputPerMinuteUsd: 0.06),
          'gpt-4o-realtime': ModelPricing(audioInputPerMinuteUsd: 0.10),
        },
        'apple': {
          'cloud': ModelPricing(),     // free
          'on-device': ModelPricing(), // free
        },
      };

      ModelPricing? priceFor(String providerId, String modelId) =>
          _table[providerId]?[modelId];

      double? calculateCostUsd({
        required String providerId,
        required String modelId,
        required LlmUsage usage,
        double? audioSeconds,
      }) {
        final p = priceFor(providerId, modelId);
        if (p == null) return null;
        var total = 0.0;
        if (usage.inputTokens > 0 && p.inputPerMillionUsd != null) {
          final nonCached = (usage.inputTokens - usage.cachedInputTokens).clamp(0, usage.inputTokens);
          total += nonCached / 1e6 * p.inputPerMillionUsd!;
        }
        if (usage.cachedInputTokens > 0 && p.cachedInputPerMillionUsd != null) {
          total += usage.cachedInputTokens / 1e6 * p.cachedInputPerMillionUsd!;
        }
        if (usage.outputTokens > 0 && p.outputPerMillionUsd != null) {
          total += usage.outputTokens / 1e6 * p.outputPerMillionUsd!;
        }
        if (usage.audioInputTokens > 0 && p.audioInputPerMillionUsd != null) {
          total += usage.audioInputTokens / 1e6 * p.audioInputPerMillionUsd!;
        }
        if (audioSeconds != null && audioSeconds > 0 && p.audioInputPerMinuteUsd != null) {
          total += audioSeconds / 60.0 * p.audioInputPerMinuteUsd!;
        }
        // Apple free entries legitimately return 0.0 — do not coerce to null.
        if (p.inputPerMillionUsd == null &&
            p.outputPerMillionUsd == null &&
            p.audioInputPerMillionUsd == null &&
            p.audioInputPerMinuteUsd == null) {
          return 0.0;
        }
        return total == 0 ? null : total;
      }
    }
    ```

## Out of scope (spec §10, enforced)

- **No cost in the Live Activity.** Do not add any field to `HelixLiveActivityAttributes`, do not render cost in Lock Screen / Dynamic Island (minimal, compact, expanded). Repeated here because the iOS Live Activity widget lives next to the recording bar conceptually and is an easy mistake.
- **No cost on the glasses HUD.** `EvenAI` / `HudController` / `BitmapHudService` must not receive any cost value.
- **No multi-currency.** USD formatting only.
- **No budgets / alerts / spending caps.**
- **No backfill** of `cost*UsdMicros` for historic conversations. Legacy rows render with the label hidden.
- **No server-side reconciliation** against the OpenAI dashboard.
